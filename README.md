###### 1 - Abrir la terminal y actualizar brew


```javascript	
sudo brew update && brew upgrade
```


###### 2 - Instalar PHP7

```javascript	
brew install php@7.2
```




###### 3 - cerrar y volver a abrir la terminal


###### 4 - por ultimo comprobar la versión de PHP con el siguiente comando


```javascript	
php -v
```
